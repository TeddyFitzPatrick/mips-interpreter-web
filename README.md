A web interpreter for the MIPS 32-bit assembly language made with TypeScript and React.

Notes:



Supported Instructions:
add
sub
and
or
xor
nor
slt
sll
srl
mult
div
mfhi
mflo
addi
andi
ori
xori
slti
beq
bne
j
jr
jal
jalr
li
la
move
mul

Limitations:
Dot derivatives not supported (e.g. .text, .data, .globl)
Memory addresses for data

Packages for development:
npm install vite
npm install tailwindcss @tailwindcss/vite
